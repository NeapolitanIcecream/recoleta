# Japanese government makes indie game devs eligible for grants up to $60k USD

## Summary
这篇文章介绍了日本经济产业省（METI）推出的 IP360 计划，其中特别允许独立游戏开发者以个人或未法人化主体身份申请最高 1000 万日元的资助，用于新 IP 开发与海外推广。其核心意义在于通过降低资助门槛，推动日本内容产业尤其是独立游戏在国际市场的竞争力。

## Problem
- 日本政府希望提升本国内容产业（如游戏、动漫）在海外市场的收入与全球竞争力。
- 传统政府资助通常要求申请者为注册公司，导致许多独立开发者和个人创作者难以获得支持。
- 对于新原创 IP 的开发、国际发行、宣传和本地化，独立团队往往缺乏足够资金与制度支持。

## Approach
- METI 推出 IP360 计划，面向新 IP 创作及国际扩展提供多种资助渠道。
- 其中“New IP Creation Support (Startup Support)”允许个人和未注册实体申请，这一点对独立开发者尤其关键。
- 资助覆盖从前期策划、正式制作到后期制作、游戏本地化和宣传推广等多个阶段。
- 申请者需提交原型和商业计划，并明确包含海外发行方案；仅限原创新 IP，续作、重制等既有 IP 项目不符合条件。
- 政府明确表示不会干预受资助作品的创意内容与创作方向。

## Results
- 单个项目最高可获 **1000 万日元**资助，约合 **6.2 万美元**。
- 补贴比例最高可覆盖 **50%** 的合规成本。
- 资金可用于 **前期开发、制作、后期制作、Localization、本地化、推广**，以及委托美术/音乐、海外展会差旅和沟通费用等。
- 日本政府设定目标：将日本内容相关的年度海外收入提升至 **20 万亿日元**（约 **1255 亿美元**），且该目标已在 **2025 年**被上升为官方国家政策。
- 文中未提供该资助计划已经带来的量化成效数据；最强的具体主张是其首次显著放宽了申请资格，使个人和非公司化独立开发者也能进入政府资助体系。
