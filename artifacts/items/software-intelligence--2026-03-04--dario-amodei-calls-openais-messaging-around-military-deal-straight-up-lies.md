# Dario Amodei calls OpenAI’s messaging around military deal ‘straight up lies’

## Summary
这篇文章不是科研论文，而是一则行业新闻，聚焦 Anthropic 与 OpenAI 在美国国防部合作条款上的公开分歧，核心争议是军方对 AI 的“任何合法用途”访问权限及其安全边界。

## Problem
- 文章讨论的问题是：AI 公司在与国防/军事机构合作时，是否应允许模型被用于**任何合法用途**，以及这种表述是否足以防止滥用。
- 这很重要，因为它涉及国内大规模监控、自动化武器等高风险场景，反映了前沿 AI 部署中的治理与安全承诺是否真实可信。
- 文章还指出，所谓“合法用途”的边界会随法律变化而变化，因此合同措辞可能不足以构成长期约束。

## Approach
- 这不是技术方法论文；其核心机制是通过**合同限制与政策红线**来约束 AI 的军用范围。
- Anthropic 的立场是：要求 DoD 明确承诺不会将其 AI 用于**国内大规模监控**或**自主武器**，否则不接受无限制访问。
- OpenAI 的立场是：其合同允许用于**所有合法目的**，并声称已在合同中明确排除国内大规模监控，因为该用途当前不属于合法使用范围。
- 争议的实质是两种治理机制的差异：**基于明确红线的限制** vs **基于现行法律定义的允许**。

## Results
- 没有提供科研实验、基准数据集或模型性能指标，因此**无量化技术结果**可总结。
- 文中最具体的数字是：Anthropic 此前已拥有一份**2亿美元**的军方合同。
- 文章称，在 OpenAI 与 DoD 达成协议后，**ChatGPT 卸载量增长了 295%**，被作为公众负面反应的信号，但未提供更完整的方法学背景或基线细节。
- 文中还提到 Anthropic 在应用商店排名升至**#2**，作为其公众形象受益的间接证据。
- 最强的实质性主张不是技术突破，而是治理层面的对比：Anthropic 声称自己拒绝“任何合法用途”的无限制访问，是为了避免滥用；Amodei 则指责 OpenAI 的安全表述是“straight up lies”。
