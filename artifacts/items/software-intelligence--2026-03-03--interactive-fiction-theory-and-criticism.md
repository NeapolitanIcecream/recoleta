# Interactive Fiction Theory and Criticism

## Summary
这不是一篇技术研究论文，而是一组关于互动小说（IF）创作、评论与社区活动的内容摘录，涵盖作者访谈、游戏节活动、作品介绍和早期 IF 历史回顾。它更偏向文化与创作讨论，而非提出可复现的新方法或实验结果。

## Problem
- 关注互动小说社区如何通过**游戏创作、评论与 game jam**促进创造力、实验性和不同 IF 社群之间的连接。
- 讨论 IF 作品如何在**叙事性**与**玩法乐趣**之间取得平衡，例如“做枯燥工作”的设定如何仍然变得“好玩”。
- 回顾早期英国业余 IF 场景与工具，说明该领域的**历史演化与社区记忆**为何重要。

## Approach
- 通过**作者与评论者对话**来呈现观点，而不是提出正式研究方法。
- 以**具体作品案例**展开讨论，如 *A Long Way to the Nearest Star*、*Cut the Sky*、*The Bat*、*The Bloody Wallpaper*、*Hampstead*。
- 用**game jam 组织经验**解释如何激发创意实验，并连接不同 IF 形式与社区。
- 结合**历史回顾**（如 *The Hobbit*、*The Philosopher's Quest*、The Quill）来讨论 IF 发展的脉络。

## Results
- 文本**没有提供量化实验结果**、数据集、基线或可比较指标，因此不存在传统意义上的研究性能提升数字。
- 最具体的成果主张主要是**作品与创作者的获奖记录**：SV Linwood 的 *Cut the Sky* 获得 SpringThing 的 **Best in Show**。
- Chandler Groover 最近发布的 *The Bat* 获得 **2024 IFComp** 冠军。
- 文本声称 game jam 能够**促进创造力、鼓励实验并搭建不同 IF 社区之间的桥梁**，但未给出量化证据。
- 对 *The Bloody Wallpaper* 的讨论提出了一个明确设计命题：即使主题是**单晚酒店劳务工作**这类琐碎劳动，作为在线游戏的 premium DLC，仍需要通过设计让玩家感到有趣。
