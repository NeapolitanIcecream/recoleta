# Trump class 'battleship' revives Navy's railgun project

## Summary
这篇文章报道了美国海军在2025年2月恢复对原型电磁轨道炮的实弹测试，显示该项目虽曾被搁置但并未彻底终止。其意义在于，轨道炮可能重新成为未来“Trump class”大型水面舰艇的关键武器，并兼具高超音速测试支撑价值。

## Problem
- 轨道炮项目此前因**供电、冷却、连续射击能力和炮管磨损**等工程难题在2021年前后被公开搁置，迟迟未能形成可海上部署的作战能力。
- 海军仍需要一种**射程远、弹药成本低、弹仓深度大**的武器，以应对海上、陆上和空中目标，甚至拦截高超音速来袭目标。
- 随着未来35,000吨级“Trump class/BBG(X)”设想推进，海军需要重新评估轨道炮是否能从实验系统转化为舰载实用武器，这关系到下一代舰艇火力结构。

## Approach
- 核心机制很简单：**轨道炮不用火药，而是用电磁力把弹丸加速到极高速度发射出去**。
- 文章披露，海军在**2025年2月于新墨西哥州白沙导弹靶场进行了为期3天的高速度发射测试**，由白沙分遣队与NSWC Dahlgren联合实施。
- 此次测试是为**NAVSEA的Joint Hypersonics Transition Office (JHTO)**开展，说明轨道炮不仅可能作为武器原型，也可能被当作高超音速相关试验平台或发射资产。
- 文章还将这次复测与未来“Trump class”舰艇计划联系起来，暗示海军可能借此重新评估现有BAE Systems原型或考虑新设计方案。

## Results
- 最明确的结果是：海军在**2025年2月完成了一次为期3天的实弹测试活动**，证明该原型轨道炮在被封存多年后**至少仍具备一定可用性**。
- 文中提到未来“Trump class”大型水面舰艇预计**排水量约35,000吨**，并被描述为可能搭载轨道炮、导弹、5英寸舰炮和激光武器的平台。
- 文章没有给出本轮测试的**炮口速度、射程、精度、发射次数、功率效率、炮管寿命**等关键定量指标，也没有提供相对导弹或传统舰炮的正式对比数据。
- 因此，本文的最强结论不是性能突破数字，而是**项目正式“复活”的信号**：轨道炮从2021年后近乎沉寂，至2025年再次实射，且已被纳入未来作战舰艇讨论框架。
- 文中还给出时间线对比：轨道炮曾在**2019年**转移至白沙靶场，**2021年**海军宣布收尾并储存硬件，而如今在**2025年**重新测试，显示战略兴趣明显回升。"
