# Kern Type: a letter spacing game

## Summary
这不是一篇研究论文，而是一个用于练习字体字距调整（kerning）的交互式网页游戏。它通过拖动/键盘微调字母间距并给出 0–100 分数，帮助用户学习排版中的字距判断。

## Problem
- 解决的问题是：如何让用户通过实践理解和训练字母间距（kerning）这一排版技能。
- 这很重要，因为字距会直接影响文字的可读性、视觉平衡和设计质量。
- 提供内容中没有研究背景、实验设定或学术问题定义，因此更像教学工具而非论文方法。

## Approach
- 核心机制非常简单：把它做成一个“letter spacing game”，让用户在界面中选择字母并左右微调位置。
- 用户可用键盘快捷键操作，如 `TAB` 切换字母、左右方向键按 1px 或 10px 移动、`ENTER` 进入下一屏。
- 系统在用户完成后给出一个分数（“Your score 0 out of 100” 的界面表明存在评分机制），并支持分享结果。
- 从给定内容看，方法本质上是“交互式练习 + 自动评分反馈”，而不是新的机器学习或算法框架。

## Results
- 给定文本**没有提供任何定量研究结果**，没有数据集、基线、用户研究、A/B 测试或统计显著性。
- 能确认的最具体结果形式是：系统会输出一个 **0–100** 的分数作为用户表现反馈。
- 文本展示了明确的交互能力：字母可按 **1px** 或 **10px** 进行微调。
- 最强的可验证主张是：这是一个用于练习 kerning 的游戏化学习工具，支持操作、评分和分享；但未给出其教学效果提升幅度。
