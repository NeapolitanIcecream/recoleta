# China Built an Underground Hypergravity Machine

## Summary
中国在杭州地下建成了超大容量超重力离心机 CHIEF1900，用极端离心力把原本需要数百到上万年的地质与工程过程压缩到几天内观测。它的意义在于为土木、地质、深海与环境研究提供比纯仿真更真实的加速物理实验平台。

## Problem
- 许多关键地质与工程过程（如土体失稳、污染物迁移、结构退化）在现实中演化极慢，往往需要数百年至上万年，难以直接实验观测。
- 大型基础设施（大坝、深海平台、高铁地基）在极端工况下的失效风险高，但全尺寸测试成本极高、周期过长，单靠计算机模拟又可能缺少真实物理数据。
- 需要一种方法在实验室中同时缩小空间尺度并压缩时间尺度，以更快、更可靠地预测真实世界中的失效与演化。

## Approach
- 核心机制很简单：把物理缩比模型安装到半径 **6.4 米** 的大型旋转臂上高速旋转，用超重力把小模型内部应力放大到等效真实大结构的水平。
- 该设备 **CHIEF1900** 可产生最高 **1,900 g-tonnes** 的载荷能力，让多吨级样品承受接近地球重力 **1900 倍** 的作用，从而加速自然过程。
- 例如，**3 米** 的大坝模型在 **100g** 下可模拟 **300 米** 真实大坝的内部应力；同理，缓慢的地质迁移、振动致损与材料劣化过程也可被加速。
- 为解决超高速旋转带来的发热与振动问题，系统被安置在地下 **15 米**、面积 **230 平方米** 的圆形舱室内，并采用真空冷却、强制通风和冷却管线来维持稳定运行。
- 设施作为共享科研平台运行，配套 **6** 个专业实验舱、**3** 个主要单元和 **18** 个飞行中装置，用于斜坡/大坝、抗震岩土、深海工程和地质过程等实验。

## Results
- 该系统宣称达到 **1,900 g-tonnes**，超过此前美国陆军工程兵团在密西西比州运行的 **1,200 g-tonnes** 纪录机型，也高于其前代 **CHIEF1300**。
- 文中给出的代表性量化结果：把通常需要**数百年至数万年**的污染物深土入渗与地下水迁移过程，压缩到 **3.65 天** 内观测。
- 文中给出的缩比等效结果：**3 米** 大坝模型在 **100g** 下可等效 **300 米** 实际结构的内部应力，用于提前发现潜在失效点。
- 先前在 CHIEF1300 上的试验已成功模拟**强震**以验证水电坝地基抗震性能，并复现 **2,000 米** 深海水压以评估海底可燃冰开采安全性。
- 先前试验还声称合成出“缺陷少、强度高”的金属合金，但本文未提供具体材料指标、对比基线或统计显著性。
- 总体上，文章的突破性主张是：该设施可构造从**毫秒到数万年**、覆盖常温到极端温度的实验环境，以获得过去难以直接获取的真实物理数据；但除容量与若干示例外，缺少系统论文式基准数据。
