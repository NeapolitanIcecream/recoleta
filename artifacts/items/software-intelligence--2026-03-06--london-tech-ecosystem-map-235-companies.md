# London tech ecosystem map (235 companies)

## Summary
这不是一篇研究论文，而是一个伦敦科技生态地图页面，汇总约235到300家公司的分布与分类。它主要提供可视化导航与筛选功能，而非提出新的算法、系统或实验结果。

## Problem
- 解决的信息问题是：伦敦科技公司与相关资源分散、难以快速浏览和按领域筛选。
- 这很重要，因为投资人、创业者、求职者和研究者需要更高效地发现本地技术生态中的公司、活动与空间。
- 但从给定内容看，它不处理软件基础模型、代码智能或多智能体软件工程等研究问题。

## Approach
- 核心机制是做一个**交互式热力/地图索引**，把伦敦科技生态中的实体集中展示在一个页面中。
- 用户可通过标签切换不同类别，例如 VC、Funding、AI、AI x Bio、Fintech、Web3、Coworking、Education、Big Tech。
- 页面还提供 Map、Events、Pubs、Share、Embed 等入口，说明其重点是信息聚合、发现与传播。
- 从最简单的角度说，它就是把“伦敦有哪些科技公司/资源”整理成可视化目录，并允许按主题筛选。

## Results
- 页面声称覆盖 **235 companies**；同时摘录中还出现 **300 so far**，两者口径不一致，可能分别表示已收录公司数与累计条目数。
- 给定内容**没有提供任何研究型定量结果**，例如准确率、召回率、排行榜名次、用户实验或与基线比较。
- 最强的具体主张是：它已经构建了一个可分享、可嵌入的伦敦科技生态地图，并支持多个行业标签筛选。
- 未见关于数据采集方法、更新频率、覆盖率验证或效果评估的数字证据。
