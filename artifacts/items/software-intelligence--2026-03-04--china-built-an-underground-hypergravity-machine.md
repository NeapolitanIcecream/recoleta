# China Built an Underground Hypergravity Machine

## Summary
中国在杭州建成了地下超重力离心机 CHIEF1900，用极端离心力把地质与工程演化过程从数百到上万年压缩到数天内观测。其意义在于为大型基础设施、地质过程和材料研究提供比纯计算模拟更直接的物理实验平台。

## Problem
- 许多地质、土木和深海工程过程演化极慢、尺度极大，直接观测往往需要数百年至上万年，实验成本和周期都不可接受。
- 仅依赖计算机模拟可能缺少关键物理数据，难以及早发现大坝、轨道、海工平台等大型结构的失效机制。
- 传统超重力设备容量有限，难以对多吨级样本施加更高载荷并稳定运行。

## Approach
- 建造地下大型超重力离心机 CHIEF1900，在半径 6.4 米的旋转臂上高速旋转重型物理模型，对样本施加最高 1,900 g-tonnes 的超重力载荷。
- 利用缩尺模型与超重力等效原理：例如 3 米模型在 100g 下可复现 300 米真实结构的内部应力状态。
- 通过把设备放在地下 15 米、230 平方米圆形舱室中，降低强振动对周边环境的影响。
- 采用真空冷却、强制通风和超大法兰配合冷却管路，控制高速旋转产生的热量，维持系统稳定。
- 将该设施作为共享科研平台，配备多个实验舱，支持坝工、地震岩土、深海工程和地质过程等多类实验。

## Results
- 设备可产生 **1,900 g-tonnes** 载荷，超过此前美国陆军工程兵团 **1,200 g-tonnes** 的纪录机型，也高于其前代 **CHIEF1300**。
- 在时间压缩方面，文中称原本需要**数百年至上万年**的自然过程可缩短到**数天**；深层土壤污染物向地下水迁移的过程可被压缩到 **3.65 天**。
- 在尺度等效方面，文中给出示例：**3 米**大坝模型在 **100g** 下可模拟 **300 米**真实大坝的应力水平。
- 先前 CHIEF1300 试验已成功模拟**强震**以验证水电大坝地基抗震性能，并复现**2,000 米深海水压**以评估海底甲烷水合物开采安全性。
- 文中还声称该平台曾合成出**缺陷更少、强度更高**的金属合金，但未提供具体材料指标、对比基线或误差范围。
- 没有给出正式论文式的标准基准数据、统计显著性或系统误差分析；最强结论是其在容量、时间压缩能力和多场景物理模拟范围上实现了工程级突破。
