# Discrete World Models via Regularization

## Summary
DWMR提出一种**无需重建器、无需对比学习**的离散世界模型学习方法，直接用正则化把图像压成布尔比特并学习动作条件转移。它面向具有组合/符号结构的环境，强调学到更信息丰富、可规划的离散状态表示。

## Problem
- 现有离散世界模型通常依赖**像素重建**来防止潜变量坍塌，但重建往往会关注视觉细节而非动作相关动力学。
- 另一类方法依赖**对比学习、奖励或价值信号**，这会增加训练机制复杂度，也不适合纯无监督、目标无关的世界模型学习。
- 作者要解决的是：**能否在完全无监督下，仅靠预测损失+适合布尔潜变量的正则化，学出非坍塌、可预测、近似符号化的离散状态与转移？** 这很重要，因为布尔状态更适合搜索、规划、精确状态匹配和组合泛化。

## Approach
- 用一个**编码器**把观测图像映射为长度为K的布尔潜变量；用一个**动作条件预测器**根据当前潜变量和动作预测下一步潜变量。
- 核心训练目标不是重建像素，而是**下一潜状态预测 + 四类正则化**：提升每个bit的方差（避免常数bit）、降低bit间相关性、降低三阶共偏度（抑制高阶耦合）、加入动作局部性先验（每步只翻少量bit）。
- 直观上，这相当于把理想表示设为“**彼此独立、接近Bernoulli(0.5)、动作只局部改动少数位**”的二进制码，从而在没有decoder的情况下避免表示退化。
- 训练时采用**两步更新**：先只用硬比特训练预测器，使其适应测试时的离散输入；再用软概率做联合反传，优化编码器和预测器的完整目标。
- 目标下一状态由**EMA目标编码器**产生，以提高离散rollout训练稳定性。

## Results
- 论文声称在两个具有组合结构的基准上，DWMR比**基于重建的替代方法**学到**更准确的表示和转移**：基准包括**MNIST 8-puzzle**与**IceSlider**，并在**干净图像**和加入**高斯噪声 N(0, 0.5)** 的设置下测试。
- 数据规模方面，8-puzzle使用**30,000**个训练转移、**6,000**验证、**6,000**测试；IceSlider使用**40,000**训练、**10,000**验证、**10,000**测试。
- 表示质量通过冻结编码器后的**线性探针**评估，指标为**mean per-cell F1-score**，分别报告直接编码（Enc.）和一步想象rollout（Im.）结果。
- 论文还明确声称：所提**两步训练程序**相比**straight-through estimator**或完全软值训练，能达到**相当或更优的准确率**。
- 此外，DWMR还可以和辅助重建器结合成**DWMR+AE / DWMR+β-VAE**，作者称这种组合会带来**进一步提升**。
- 但在给定摘录中，**没有提供最终主表的具体数值结果**（如各方法在各数据集上的F1均值/方差、相对提升百分比），因此无法准确列出“超过基线多少点”的定量结论。
