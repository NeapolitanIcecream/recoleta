Repository-Scale Coding Agent Governance

代码代理的采用正在仓库边界上获得更具体的支持：更小的启动上下文、代码地图、使用日志、明确的工作流文件，以及保护领域词汇的审查做法。具体工作是先把 agent 行为做成可测、可重复，再让团队把它用在更大的代码变更上。
