Code Change Control

最清楚的短期变化都在执行层面。编码代理产品需要在运行中有明确的 token 控制，仓库规模生成在项目变大后需要按依赖顺序工作流，而维护团队可以为把需求链接到代码、并带上更小上下文和可见证据的追踪层找到理由。
