# Denmark is set to explore if gastronomy can be recognized as an art form

## Summary
这篇文章讨论丹麦是否应正式将高级餐饮/美食学认定为一种艺术形式，核心背景是哥本哈根部分米其林餐厅正把食物、表演、音乐与社会议题结合成沉浸式体验。它不是技术论文，而是一则关于文化政策、艺术定义与餐饮行业边界的报道。

## Problem
- 要解决的问题是：**美食学，尤其是高端、观念化、沉浸式餐饮，是否应被法律和文化制度正式视为“艺术”而非“工艺”**。
- 这之所以重要，是因为一旦被认定为艺术，厨师与餐饮项目可能获得**国家补贴、基金会资助和更高文化地位**，改变行业资源分配与社会认知。
- 争议点在于：支持者认为食物可以像绘画或音乐一样承载表达；反对者认为食物的核心是被消费，功能属性使其与艺术本质不同。

## Approach
- 丹麦文化部长在2025年1月表示，政府将**探索**是否正式承认美食学为艺术形式；若推进，最终需要丹麦**179席议会投票**完成从“craft”到“art”的重新分类。
- 支持这一主张的餐厅实践路径，是把餐饮从单纯菜品提供扩展为**沉浸式叙事媒介**：例如 Alchemist 用投影、表演、音乐、空间设计和食物共同表达海洋塑料污染、国家监控、动物福利等议题。
- 核心机制可以简单理解为：**把“食物”当作表达思想的媒介，而不仅是可食用商品**；厨师像艺术家一样设计完整体验，让用餐者在进食同时被引导反思社会问题。
- 文章也呈现了反方机制：批评者认为即便高级餐饮能制造体验，它仍与艺术不同，因为艺术的唯一目的可以是表达，而食物还必须满足“被食用”的功能。

## Results
- 文章**没有提供实验、数据集或学术基准上的定量结果**，因此不存在传统研究意义上的性能突破。
- 最强的具体进展是政策层面：丹麦政府已在**2025年1月**公开表示将探索此议题；若落实，丹麦可能成为**首个**在法律层面将烹饪/高级美食放到与传统艺术相近地位的国家。
- 支持案例方面，Alchemist 餐厅于**2019年**开业，**2025年被评为世界第5佳餐厅**，并拥有**2颗米其林星**；其用餐体验包含约**50个“impressions”**，作为“美食可成为艺术”论点的代表性样本。
- 行业背景方面，丹麦这个约**600万人口**国家拥有**37家米其林星级餐厅**，显示其具备推动此类文化认定的餐饮生态基础。
- 文章同时强调了阻力：政策仍处于**exploratory phase（探索阶段）**，且可能受到**3月24日大选**影响；艺术界与部分厨师担忧这会带来**艺术资助竞争加剧**与概念混淆。
