# I'll probably never use Windows

## Summary
这不是一篇机器人或机器学习研究论文，而是一篇关于 Windows 发展方向的观点文章。作者认为现代 Windows 正在削弱用户对个人电脑的控制权，并因隐私、云依赖和系统臃肿问题而不再值得使用。

## Problem
- 文章讨论的问题是：现代 Windows 正从“用户可控的本地操作系统”转向“受厂商和云服务控制的平台”。
- 作者认为这很重要，因为它会带来隐私风险、系统不稳定、强制账户绑定、广告化和用户自主权下降。
- 文中还强调，这种变化可能让政府或大公司更容易追踪、审查或错误处置普通用户的数据与行为。

## Approach
- 这篇文章没有提出技术方法、算法或实验系统；核心机制是作者基于个人观察与产品演进做出的批判性论述。
- 作者将 Windows 7 视为最后一个“真正优秀”的版本，并把 Windows 8/10/11 的变化概括为：UI 更混乱、测试更多转移给终端用户、越来越依赖云与 Microsoft 账户。
- 文章用最简单的话说，其核心论点是：Windows 不再只是“你的电脑上的操作系统”，而更像是“连接到微软服务并受其影响的终端”。
- 作为替代方案，作者建议重视隐私的用户考虑 Linux；若追求更完整的商用体验，也可考虑 macOS。

## Results
- 没有提供任何量化实验结果、数据集、指标或与基线的正式比较。
- 最强的具体主张包括：Windows 10 和 11 的更新“使许多 PC 无法使用”，但文中没有给出数量、来源或统计证据。
- 作者声称 Windows 已变成“广告平台”，并且在初始设置中更强制要求 Microsoft 账户登录，但未提供可重复验证的实验测量。
- 文章还声称 Linux 游戏兼容性因 Proton “显著改善”，但同样没有给出兼容率、性能数字或对比测试。
