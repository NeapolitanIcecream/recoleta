# The Sandboxed Open-Source Agent that is 70% cheaper than E2B

## Summary
Coasty 是一个在隔离沙箱中运行的开源自治代理/“AI 劳动力”平台，主打用浏览器和真实软件自动执行业务运营任务。文案核心主张是以更低成本替代部分人工运营团队，并在 OSWorld 基准上取得领先表现。

## Problem
- 企业在市场、销售、运营、QA、报销处理等重复性软件操作上需要大量人工，招聘、培训和管理成本高。
- 传统聊天机器人不能真正完成端到端的软件操作，而现实业务需要代理能打开网页、填写表单、提取数据并持续纠错。
- 这很重要，因为如果代理能稳定执行真实计算机任务，就可能显著降低运营人力成本并提升 24/7 自动化能力。

## Approach
- 核心机制是让 AI 代理像人类员工一样直接操作真实软件：打开浏览器、导航应用、点击、输入、抓取信息并完成任务。
- 系统运行在隔离沙箱中，强调会话隔离与数据安全，适合执行真实业务操作。
- 代理具备“自我纠错”执行能力：遇到误点或死路时自动调整路径继续完成任务。
- 平台记录每一次点击、击键和决策，提供完整审计追踪，方便人工复核与后续部署。
- 使用方式被描述为“指派一个 operation”，然后让代理自主完成，例如外联、发票处理、QA 测试或报告生成。

## Results
- 成本主张：标题称其“比 E2B 便宜 70%”，但摘录中未给出实验设置、价格公式或直接对照明细。
- 基准成绩：宣称在 **OSWorld** 上达到 **82% success rate**，并称为 **#1 in the World**；摘录未提供具体拆分任务、评测协议或与第二名/特定基线的数值差距。
- 人力替代主张：称“3-person operations team costs $12,000+/mo”，而 Coasty 可承担“the same workload 24/7”；但未给出 workload 定义或严格对照实验。
- 部署效率主张：宣称可在 **60 seconds** 内部署。
- 任务规模示例：可每周进行 **10 markets** 的竞品研究、维护 **500-lead pipeline**、对 **100 leads** 执行外联；这些是能力展示，不是严格基准结果。
