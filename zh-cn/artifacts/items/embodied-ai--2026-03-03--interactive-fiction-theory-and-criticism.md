# Interactive Fiction Theory and Criticism

## Summary
这不是一篇机器人或机器学习研究论文，而是一个关于**互动小说（Interactive Fiction, IF）理论、批评、作者与作品讨论**的网页摘录。内容主要介绍作者、作品、社区活动与游戏文本评论，因此与用户关注的机器人基础模型主题几乎无关。

## Problem
- 该文本**没有提出一个明确的科学或工程问题**，而是围绕互动小说创作、评论与社区建设展开。
- 从内容上看，它关注的是**如何通过 game jam 促进创造力、实验性以及不同 IF 社群之间的连接**。
- 这类讨论对数字叙事与游戏文化有意义，但**不涉及 embodied AI、VLA、world model 或机器人数据扩展问题**。

## Approach
- 通过**作者介绍与作品列举**来建立互动小说领域的背景与代表人物。
- 通过**访谈/聊天形式**讨论举办 game jams 如何促进创意实验和社群桥接。
- 通过对具体作品（如 *The Bloody Wallpaper*、*Hampstead*）的**评论与回顾**，展开理论和批评层面的讨论。
- 采用的是**文学/游戏批评与社区观察**的方法，而非实验、建模或算法设计。

## Results
- 文本**没有提供任何定量实验结果、数据集、基线或性能指标**。
- 最具体的可验证信息是作品与奖项：*Cut the Sky* 获得 **SpringThing Best in Show**，*The Bat* 获得 **2024 IFComp**。
- 文本声称 game jams 可以**促进创造力、鼓励实验，并连接不同 IF 社群与形式**，但未给出数字证据。
- 还提供了对英国早期 IF 场景及作品（如 *The Hobbit*、*The Philosopher's Quest*、*The Quill*、*Hampstead*）的历史性回顾，但这不是研究结果。
