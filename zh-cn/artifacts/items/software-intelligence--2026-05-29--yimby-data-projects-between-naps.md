# YIMBY data projects, between naps

## Summary
## 摘要
这篇文章介绍了三个用 Claude Code 在照看孩子的短暂间隙里做出来的小型 YIMBY 公共数据项目。核心观点是，编码代理让一个人能在几小时内把公共本地数据做成可分享的网页，用来讨论住房议题。

## 问题
- 圣马特奥的本地住房会议里有关于审批速度、学校入学人数和面向公共交通的分区容量的经验性说法。
- 居民需要带出处、能分享的本地证据，在听证会上直接使用，不必等记者、顾问、资助项目或 GIS 专家。
- 这个问题很重要，因为本地土地使用争论常常会影响住房供应、学校规划和交通站点周边分区。

## 方法
- 作者用 Claude Code 查找公开数据集、写代码，并把小型网页项目发布到 GitHub Pages。
- 一个项目用加州住房与社区发展办公室的年度进展报告数据，对半岛 22 个城市按建设审批速度、RHNA 进展、ADU 合规情况和按管辖区计算的摩擦分数进行排名。
- 第二个项目把圣马特奥县各学区按 2021-2025 年发放的住房许可数量和自 2018 年以来的入学人数变化进行散点图展示。
- 第三个项目在符合条件的交通站点周边绘制 SB 79 的地块级容量，使用的密度规则是：距站点 200 英尺内每英亩 160 户，四分之一英里内每英亩 120 户，半英里内每英亩 100 户。

## 结果
- 文章提到 3 个公开的公民数据项目，每个都在几次短时工作间隙里完成。
- 许可追踪器覆盖 22 个半岛城市，并把它们的审批速度、RHNA 进展和 ADU 合规情况做成可排序结果，便于公开比较。
- 入学人数项目发现，2021-2025 年住房许可数量与自 2018 年以来入学人数变化之间的关系是平坦到略为负相关；文中展示的每个圣马特奥县学区都低于零增长线。
- 作者称，红木城和南旧金山发放了数百个住房单元的许可，但入学人数下降幅度与 Hillsborough 和 Portola Valley 相近，而后两地几乎没有新建住房。
- SB 79 地图按 160、120 和 100 户/英亩三个密度区间，估算了交通站点周边新增住房容量。
- 摘要没有给出基准化的软件工程指标、用户研究，也没有把代理工作流和非代理工作流做对照比较。
