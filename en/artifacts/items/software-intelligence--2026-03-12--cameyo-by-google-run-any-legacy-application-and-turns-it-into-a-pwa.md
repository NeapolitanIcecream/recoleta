# Cameyo by Google: Run any legacy application and turns it into a PWA

## Summary
This is not an academic paper, but a product/solution overview for Cameyo by Google. It argues that traditional legacy client applications can be delivered as web apps/PWAs through virtualization, thereby supporting a more secure, cross-device, web-first work environment.

## Problem
- Enterprises still rely on a large number of legacy client applications, which hinders migration to a **web-first** way of working, ChromeOS, and modern collaboration suites.
- Traditional virtual desktop solutions are “anchored to the past” and are not ideal in terms of user experience, operations, and integration with modern browsers.
- IT needs to simultaneously satisfy security, cross-device access, business continuity, and legacy app compatibility, which is critical for digital transformation.

## Approach
- Use **Virtual App Delivery** to replace traditional virtual desktops, delivering the client applications businesses depend on as **web apps/PWA**.
- Integrate application access into the **Chrome Enterprise** secure enterprise browser, enabling users to access apps “anytime, from any device.”
- Work together with **Google Workspace** so organizations can retain access to critical client applications while adopting a more collaborative web-first productivity suite.
- Integrate with **ChromeOS** so that when moving to a more secure, web-based operating system, legacy applications can still be used seamlessly through the browser.
