# Show HN: Paste a URL and watch multiple AI models redesign it side-by-side

## Summary
This is a multi-model AI design tool for website redesigns: users paste a URL and enter their goals, the system has multiple models generate redesign proposals in parallel, and then provides visual editing and code export. Its value lies in automating repetitive front-end design/build work and helping users make design decisions faster through side-by-side comparison.

## Problem
- Website redesigns usually require a large amount of repetitive front-end design and implementation work, which is time-consuming and depends on repeated manual trial and error.
- A single AI solution often provides only one interpretation, making it hard to compare how different models approach the same requirement from a design perspective.
- Business goals such as improving conversion, modernization, or usability require design tools that can iterate quickly around clear objectives, rather than merely “generating attractive pages.”

## Approach
- The user enters the existing website URL and redesign intent/prompt, and the system uses this to start the generation process.
- Multiple AI models (the text cites Claude, GPT, Gemini, etc.) run in parallel, each independently producing a website redesign proposal.
- The platform displays the results side by side, making it easy for users to directly compare how different models interpret the same brief and their stylistic differences.
- Users can fine-tune components, typography, colors, and more in Shuffle through drag-and-drop editing, without writing CSS by hand.
- The final output can be exported as production-ready code, supporting Next.js, Laravel, WordPress, or plain HTML/CSS.
