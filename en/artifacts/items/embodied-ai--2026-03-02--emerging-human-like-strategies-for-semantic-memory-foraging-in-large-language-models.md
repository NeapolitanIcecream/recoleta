# Emerging Human-like Strategies for Semantic Memory Foraging in Large Language Models

## Summary
This paper studies whether the “semantic memory foraging” of large language models in the semantic fluency task exhibits human-like clustering and switching strategies. Using mechanistic interpretability methods, the authors find that these convergent/divergent patterns not only appear in model outputs, but can also be read out from internal representations in the middle-to-late layers.

## Problem
- The paper aims to answer: **When retrieving concepts from semantic memory, do LLMs use identifiable, human-like search strategies**, especially the “clustering (convergent)” and “switching (divergent)” seen in the semantic fluency task?
- This matters because if these mechanisms can be localized, we can more rigorously understand whether LLMs are cognitively “aligned” with humans or merely superficially similar while relying on different mechanisms, and it also provides a foundation for later behavioral steering and representation engineering.
- It also connects to a core question in mechanistic interpretability: which internal layers and representations actually support this seemingly “cognitive” behavior.

## Approach
- The authors use the classic **Semantic Fluency Task (SFT)**: ask a person or model to produce as many animal names as possible in sequence, then use category norms to label consecutive word pairs as either “same-category continuation” (non-switch / convergent) or “cross-category switch” (switch / divergent).
- On the data side, they compare **699 human sequences** with an equal number of task sequences generated by the **Llama-3 family (1B/3B/8B/70B)**; after filtering, they analyze **681 human sequences** and **2285 LLM sequences**.
- At the output level, the authors divide the vocabulary into a “current-category token set” and a “cross-category token set,” examine how the probability mass of these two token groups changes around switch events, and use the **logit lens** to trace when this difference emerges across layers.
- At the representation level, they apply **PCA + logistic regression probe** to the residual stream to test whether switch vs. non-switch can be directly classified from middle-layer activations; they also construct contrastive prompt data with neutral, convergent, and divergent conditions to strengthen mechanistic readout.

## Results
- The category transition matrices of humans and LLMs are highly correlated: **Spearman ρ = 0.701, p < 0.001**, indicating similar overall semantic search patterns.
- However, LLMs switch categories less often than humans: humans have an average **switch ratio = 0.55**, while LLMs average **0.40**, with a significant difference (**Mann-Whitney p < 0.0001**); the authors therefore argue that LLMs are more inclined to more exhaustively “drain” the same semantic cluster.
- In the 70B model, the output distribution changes significantly near switch points: from relative position -1 to 0, **within-category d = -0.158**, **between-category d = 0.144**, **actual sequence d = -0.184**, all with **p < 0.001**, showing that when a switch occurs, the model’s relative preference for “cross-category words” increases.
- The logit lens shows that the signal distinguishing switch vs. non-switch becomes clearly visible in the **middle-to-late layers (approximately layer > 40)**; in late layers, during non-switches **within-category prob = 0.0035** and **between-category prob = 0.0005**, whereas during switches this becomes **0.0008 vs. 0.0007**, still with a significant difference (**Mann-Whitney, p < 0.001**).
- When using only residual-stream representations from real human sequences with a linear probe, performance is weak: **AUROC = 0.57** (Llama-3.3-70B); by contrast, classification using the output distribution reaches **AUROC = 0.751**.
- On the authors’ constructed contrastive prompt data, the switching mechanism in the residual stream can be read out strongly: under a linear probe, the 70B model reaches **AUROC = 0.96 (neutral) / 0.98 (convergent) / 0.97 (divergent)**; at the same time, larger models generally achieve better readout performance.
