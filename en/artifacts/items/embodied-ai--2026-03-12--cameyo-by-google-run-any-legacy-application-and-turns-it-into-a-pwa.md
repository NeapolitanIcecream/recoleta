# Cameyo by Google: Run any legacy application and turns it into a PWA

## Summary
This is not a research paper, but a product marketing page describing how Google's Cameyo virtual app delivery solution delivers traditional client/legacy applications through the browser in a PWA-like way. Its core value proposition is to reduce reliance on traditional virtual desktops and help enterprises move toward a web-first workplace.

## Problem
- Many enterprises still depend on legacy client applications and cannot easily migrate directly to pure web/cloud-native workflows.
- Traditional virtual desktop solutions are complex and heavyweight, potentially increasing IT operations burden and harming end-user experience.
- Enterprises want to improve security, cross-device access, and the modern workplace experience while continuing to use critical legacy applications.

## Approach
- Provide **Virtual App Delivery**: deliver the traditional applications a business relies on as web apps accessible through the browser, rather than delivering an entire virtual desktop.
- Integrate with **Chrome Enterprise** to embed virtualization access capabilities into a secure enterprise browser, allowing users to access applications “anytime, from any device.”
- Integrate with **Google Workspace** so organizations can continue to access critical client-based applications seamlessly while adopting a web-first collaboration suite.
- Integrate with **ChromeOS**, claiming that legacy applications can still be used on a more secure, web-first operating system.
