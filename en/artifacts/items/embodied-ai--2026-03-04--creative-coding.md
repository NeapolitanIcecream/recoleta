# Creative Coding

## Summary
This passage introduces the practice of "creative coding": using programming to create expressive and artistic works rather than prioritizing functionality. It reads more like a conceptual encyclopedia entry than a research paper, so it does not propose or validate any new technical method.

- It tries to answer the question: how can programming be used as a medium for artistic expression rather than merely as a tool for functional software development.
- This matters because it relates to the creative methods and cultural positioning of contemporary digital art, interaction design, live visuals, installation art, and generative creation.
- The text also points to a deeper issue: in computer art, the role of code itself is often obscured by the "interface/user" perspective, and the relationship between code and creative intent needs to be understood more carefully.
