# Spitting Out the Agentic Kool-Aid

## Summary
This essay argues that agentic coding tools can create unhealthy attachment, compulsive use, and pressure on open source maintainers. It matters to software teams because code agents are moving into daily development faster than norms for safety, ownership, and attention.

## Problem
- Agentic coding tools such as Claude Code, OpenClaw, and Pi can make developers feel more productive while also increasing compulsion, FOMO, and parasocial attachment.
- Open source projects face slop PRs, spam, and maintainer load as code agents make it cheap to generate contributions.
- The author links software agents to a wider concern: screen-based tools and AI companions may replace human bonds and pull more of daily life into corporate systems.

## Approach
- The author used Claude Code with Opus 4.5 on a side project, then compared the experience with public comments from Peter Steinberger, Mario Zechner, Armin Ronacher, and the Center for Humane Technology.
- The method is first-person evaluation: use the tool for intense coding sessions, observe mood, attention, work patterns, and failure modes, then decide what to keep or reject.
- The essay treats agentic coding as a human-AI interaction problem, not only a software productivity problem.
- The practical response is behavioral: reduce agent use, separate work screens from private life, try analog days, and launch a print magazine called Gift.

## Results
- The excerpt reports no benchmarked technical results, no dataset, and no measured coding accuracy or productivity gain.
- The author used Claude Code for three 12+ hour days, then added a 2-day deadline to ship an MVP, but abandoned the MVP after bugs and discomfort.
- OpenClaw is tied to November 24, the same day Anthropic released Opus 4.5, and the essay presents its rapid spread as a social signal rather than a measured result.
- The strongest concrete claim is experiential: after about 5 days of heavy use, the author judged agentic coding too psychologically costly for personal use.
- The proposed mitigation has 4 levels: phone out of the bedroom, Analog Sunday, Screen Time/Bean Time, and an Analog Economy.
