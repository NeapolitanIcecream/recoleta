# Plasma Bigscreen – 10-foot interface for KDE plasma

## Summary
Plasma Bigscreen is an open-source large-screen Linux interface for TVs, HTPCs, and set-top boxes, emphasizing customizability, an open ecosystem, and privacy friendliness. It is not a research model or algorithm paper, but rather a product/platform introduction whose core value is providing an open alternative to closed TV systems.

## Problem
- Existing TV and set-top box platforms are typically **closed ecosystems**, lacking openness, controllability, and verifiability.
- These systems often have issues around **privacy and user trust**, making it difficult for users to truly control the device and software stack.
- TV scenarios require an interaction model suited to a “10-foot interface”: usable from a distance, compatible with inputs like remotes and controllers, while also supporting app launching, settings management, and multitasking.

## Approach
- It uses a **TV-oriented large-screen Linux desktop environment**: combining open technologies such as KDE Plasma, Qt, KWin, and Kirigami into a TV UI suited to the living-room setting.
- The core idea can be understood simply as **repackaging the Linux desktop into a TV-friendly interface**, allowing users to perform everyday tasks from the couch using a remote, CEC, controller, keyboard and mouse, or phone.
- It provides **TV-friendly settings** and a one-button home overlay, making app search, settings access, returning home, and task switching better suited to large-screen, distance-based interaction.
- It connects to the existing app ecosystem through Linux distribution package managers and Flathub, and supports apps such as Steam, Kodi, Jellyfin, and VacuumTube, emphasizing extensibility and customization.
- It is based on **fully open-source, community-driven development**, allowing distributions and vendors to adopt it directly, while also letting users freely modify and deploy it.
