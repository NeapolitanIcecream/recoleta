Authors

Summary

IPD is an offline reinforcement learning method aimed at enabling Transformer-style sequential policies to do more than merely imitate offline data, so they can learn better decisions from “imagined planning.” It distills a world model, value function, and high-quality virtual trajectories generated by MPC into the policy training and inference pipeline.

Problem

  • Existing Decision Transformer-style offline RL methods mainly perform conditional imitation and struggle to “stitch” together multiple suboptimal segments into a better policy.
  • Pure offline data often contains many suboptimal trajectories, and out-of-distribution actions can lead to value overestimation and unstable decisions.
  • This matters because in real-world settings, online exploration is costly and risky; if offline RL cannot extract better behavior from static data, its practical deployment value is limited.

Approach

  • First, learn from offline data a world model with uncertainty estimation and a quasi-optimal value function / Q-function trained in an IQL-style manner to reduce out-of-distribution overestimation.
  • Use the value function to compare returns from real trajectories and “imagined trajectories,” identifying the suboptimal states/segments most worth rewriting.
  • For these segments, use MPC to plan in the learned world model and generate better virtual rollouts; then apply an uncertainty threshold filter, keeping only reliable samples to add to the augmented dataset.
  • When training the Transformer policy, do not just fit the augmented sequences; also add a value-guided objective to distill knowledge from dynamic programming and MPC into the policy.
  • At inference time, use the learned quasi-optimal value function instead of manually specified return-to-go as the conditioning signal to improve stability and performance.

Results

  • The paper claims that on the D4RL benchmark, IPD “significantly outperforms” various SOTA value-based and transformer-based offline RL methods.
  • The abstract and introduction provide qualitative conclusions: consistent gains across diverse tasks, and ablations validating the contributions of modules such as MPC data augmentation, value-guided action imitation, and return-to-go prediction.
  • The paper also claims to analyze the scaling law relationship between imaginary data augmentation volume and performance.
  • The currently provided excerpt does not include specific numerical results, so precise scores, percentage improvements, corresponding baselines, or task-level comparison numbers cannot be listed.
