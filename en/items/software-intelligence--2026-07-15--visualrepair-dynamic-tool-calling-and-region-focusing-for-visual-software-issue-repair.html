VisualRepair: Dynamic Tool Calling and Region Focusing for Visual Software Issue Repair

Summary

VisualRepair targets automated repair of software defects that are conveyed through heterogeneous visual artifacts such as UI screenshots, GIFs, IDE snapshots, and text images. It combines type-specific visual tools with dynamic region focusing and reports 196 resolved issues on SWE-bench Multimodal test, 10 more than the best baseline.
