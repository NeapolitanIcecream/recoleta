## [0.1.0] - 2026-03-16

### Added

- local-first research pipeline backed by SQLite
- multi-source ingestion for arXiv, Hacker News RSS, Hugging Face Daily Papers,
  OpenReview, and RSS feeds
- structured LLM analysis with topic scoring and relevance filtering
- local Markdown output with optional Obsidian and Telegram delivery
- trend briefs, idea briefs, browser-rendered PDFs, and static site export
- Docker runtime images, Compose workflow, healthcheck, and JSON workspace stats
- starter presets for `agents-radar`, `robotics-radar`, and `arxiv-digest`
- repo-return CTAs on the generated static site
- a shorter README plus dedicated usage and first-output guides
- GitHub contribution templates, release draft workflow, and launch handoff docs
- sample output screenshots and preset-to-demo mapping for first-run validation

### Notes

- `0.1.0` is the first public release baseline for the current repo surface.
