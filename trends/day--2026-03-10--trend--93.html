Overview

今天的研究主题很集中：智能体正在从“会说”走向“会做”，而且评测也在从“能答题”走向“能工作”。

一条清晰主线是，研究者开始减少对“大模型参数量”和“超长上下文”的依赖，转而优化智能体的决策过程。ATLAS 认为关键不只是给模型更多上下文，而是让它学会何时取什么信息、如何按结构调用工具。ACT 则进一步把“判断哪个动作更优”直接做成训练目标。AutoResearch-RL 更激进，它让代理自己改训练代码、自己跑实验、自己根据结果继续搜索。这些工作放在一起看，说明强化学习（RL，强化学习）和闭环反馈，正在重新成为智能体能力提升的核心抓手。

另一条主线是，基准明显变难，也更贴近高价值场景。$OneMillion-Bench 面向法律、金融、医疗、工业和自然科学等专业任务，不再只看最后答案，而是看证据、规则和可执行性。OfficeQA Pro 进一步把问题推进到企业文档环境，要求模型在大量历史金融材料、表格和数字中做端到端有依据推理。从结果看，这个方向很有警示意义：当前前沿系统离“企业级可靠”还很远。

还有一条值得注意的变化，是智能体开始从被动响应走向主动交互。PIRA-Bench 关注 GUI 智能体能否从连续屏幕轨迹中主动推断用户意图。CUDAMaster 则把代码优化放进真实编译、执行和 profiling 反馈回路中。两者都表明，模型的价值越来越依赖它是否能在环境中持续行动，而不是一次性给出漂亮答案。

整体来看，今天的样本虽少，但方向非常一致：训练上更重决策质量，评测上更重真实工作，系统上更重环境闭环。这意味着下一阶段的竞争点，可能不是谁能把上下文堆得更长，而是谁能在复杂流程里更稳定地做对事。

Clusters

智能体训练从“会回答”转向“会决策”

当天最强主线是“让智能体更像可执行系统，而不是更长上下文的聊天模型”。一类工作聚焦训练方法：ATLAS把上下文选择和工具编排本身变成可学习决策，ACT则直接训练模型判断哪个动作更优。另一类工作聚焦开放式实验循环：AutoResearch-RL让代理持续改代码、跑实验、再根据结果更新策略。这些工作共同指向一个变化：能力提升的重点，正在从单次回答质量转向多步决策质量。

Representative papers

  • Scaling Agentic Capabilities, Not Context: Efficient Reinforcement Finetuning for Large Toolspaces
  • Agentic Critical Training
  • AutoResearch-RL: Perpetual Self-Evaluating Reinforcement Learning Agents for Autonomous Neural Architecture Discovery

基准从学术题目走向高价值真实工作

第二条主线是评测标准明显升级。$OneMillion-Bench 把目标抬到法律、金融、医疗等高价值专业任务，强调证据检索、规则应用和可执行决策。OfficeQA Pro 则把企业文档推理做得更硬核，要求模型在近百年金融文档、表格和数值中完成端到端有依据推理，而且现有前沿系统表现仍很低。这说明社区不再满足于短题得分，而是在逼近真实工作的可靠性门槛。

Representative papers

  • $OneMillion-Bench: How Far are Language Agents from Human Experts?
  • OfficeQA Pro: An Enterprise Benchmark for End-to-End Grounded Reasoning

智能体开始追求主动交互与环境闭环

第三条主线是“主动性”和“闭环执行”继续外扩。PIRA-Bench 不再只测 GUI agent 是否能响应指令，而是看它能否从连续屏幕轨迹中主动判断用户意图并给建议。CUDAMaster 则把代码生成、编译、运行和 profiling 接到一起，让 LLM 在真实反馈中优化 CUDA 内核。两者都强调一件事：模型价值越来越取决于它能否在环境中持续感知、行动并根据反馈修正。

Top-7 必读

  1. Scaling Agentic Capabilities, Not Context: Efficient Reinforcement Finetuning for Large Toolspaces —— 代表“能力不靠堆上下文，而靠学会取用上下文和结构化执行工具”的训练思路。
  2. OfficeQA Pro: An Enterprise Benchmark for End-to-End Grounded Reasoning —— 直接展示企业文档推理的真实难度，是当天最能说明能力缺口的基准论文。
  3. Agentic Critical Training —— 把“判断更优动作”作为训练目标，适合关注代理反思与决策训练的人。
  4. $OneMillion-Bench: How Far are Language Agents from Human Experts? —— 把评测目标抬到高价值专业劳动，适合判断智能体离专家工作还有多远。
  5. Making LLMs Optimize Multi-Scenario CUDA Kernels Like Experts —— 展示 LLM 在真实工程反馈回路里优化 GPU kernel 的潜力。
  6. AutoResearch-RL: Perpetual Self-Evaluating Reinforcement Learning Agents for Autonomous Neural Architecture Discovery —— 关注“无人值守研究循环”与自动模型研发的人应优先阅读。
  7. PIRA-Bench: A Transition from Reactive GUI Agents to GUI-based Proactive Intent Recommendation Agents —— 代表 GUI agent 从被动执行迈向主动意图推荐的新方向。
