Summary

这不是一篇研究论文，而是一页产品宣传内容，介绍 Google 的 Cameyo 虚拟应用交付方案如何把传统客户端/遗留应用通过浏览器以类似 PWA 的方式交付。其核心价值主张是减少传统虚拟桌面依赖，推动企业向 web-first 办公迁移。

Problem

  • 许多企业仍依赖遗留客户端应用，难以直接迁移到纯 Web/云原生工作流。
  • 传统虚拟桌面方案复杂、沉重，可能增加 IT 运维负担并影响终端用户体验。
  • 企业希望在提升安全性、跨设备访问能力和现代化办公体验的同时，继续使用关键旧应用。

Approach

  • 提供 Virtual App Delivery：把企业依赖的传统应用交付为可通过浏览器访问的 Web 应用，而不是整套虚拟桌面。
  • 与 Chrome Enterprise 集成，把虚拟化访问能力嵌入安全企业浏览器中，使用户可“随时、任意设备”访问应用。
  • 与 Google Workspace 集成，让组织在采用 web-first 协作套件时，继续无缝访问关键客户端应用。
  • 与 ChromeOS 集成，主张在更安全、web-first 的操作系统上仍可使用遗留应用。
