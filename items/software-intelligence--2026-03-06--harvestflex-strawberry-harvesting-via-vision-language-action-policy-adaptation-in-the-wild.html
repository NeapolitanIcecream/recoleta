HarvestFlex: Strawberry Harvesting via Vision-Language-Action Policy Adaptation in the Wild

Summary

本文研究将开源视觉-语言-动作（VLA）策略迁移到真实温室草莓采摘这一长时程、强遮挡、易损伤的任务上，并构建了一个仅用三视角RGB的端到端闭环系统。作者声称，在不到4小时真实示教数据下，微调后的VLA策略已能实现较可靠的真实采摘。

Problem

  • 目标问题：让机器人在真实温室台架环境中完成草莓采摘与放置，这是一项长时程、闭环、非结构化且接触敏感的操作任务。
  • 难点在于叶片遮挡、镜面反光、光照变化、接触后的状态变化会让传统“感知-规划-控制”模块化流水线误差累积，降低跨农场泛化和鲁棒性。
  • 该问题重要，因为草莓采摘仍高度依赖人工，存在季节性劳动力紧张、成本上升、供工不稳定等现实痛点，而自动化系统还必须兼顾成功率、效率和果实损伤率。

Approach

  • 核心方法很简单：把三路RGB图像 + 机器人状态 + 语言指令直接输入VLA策略，让模型直接输出机械臂动作和气泵夹爪指令，省去深度点云、显式几何标定和手工状态机。
  • 系统搭建在HarvestFlex平台上，使用两路固定场景相机 + 一路腕部相机，以增强目标选择、接近和接触阶段的可观测性，同时刻意避免依赖深度和精确校准。
  • 数据方面，作者通过VR遥操作采集了3.71小时、227个episode、491次有效采摘尝试的真实示教，用于微调三种开源VLA模型：π0、π0.5、WALL-OSS，并比较全量微调与LoRA。
  • 部署方面，作者提出异步推理-控制解耦：推理线程批量产生活动作块，实时控制线程以30 Hz固定频率执行，从而减少同步推理带来的控制抖动和接触窗口错失。

Results

  • 在统一的50次真实温室试验协议下，最佳结果来自π0.5 + 全量微调 + 6 epochs：成功率 74.0%，Success Score 82.6，32.6 s/pick，损伤率 4.1%。
  • 同样是6 epochs全量微调，其他模型结果为：WALL-OSS 68.0% SR / 78.8 SS / 46.3 s / 3.9% DR；π0 60.0% SR / 72.6 SS / 38.4 s / 4.2% DR。说明π0.5总体完成度最高，WALL-OSS损伤略低。
  • LoRA整体弱于全量微调：在π0.5, 6 epochs下，LoRA为64.0% SR / 73.6 SS / 38.3 s / 3.8% DR，相比全量微调少了10个百分点成功率，但损伤率接近。
  • 随训练轮数从2→4→6 epochs增加，各模型成功率普遍上升、周期时间下降。例如π0.5全量微调从30.0% → 50.0% → 74.0% SR，周期时间从44.2s → 40.7s → 32.6s。
  • 数据规模方面，作者强调仅用3.71小时真实数据就实现了“非平凡”的闭环采摘能力；同时也明确指出瓶颈仍在于近距离可观测性丢失和接触动力学失配。
  • 论文摘要还声称异步部署优于同步部署，但给定摘录中未提供对应的具体对比数字；最强的具体结论是异步推理控制解耦进一步提升了真实部署表现。
