- `v0.1.0`

Suggested release title:

- `v0.1.0: local-first AI research radar baseline`

## Suggested checklist

1. Confirm `README.md` links and badges are current.
2. Confirm `CHANGELOG.md` and `docs/releases/v0.1.0-draft.md` match.
3. Reuse `docs/releases/v0.1.0-launch-kit.md` for the release body or launch post.
4. Upload the social preview PNG.
5. Update the repo About section and website.
6. Add the chosen GitHub topics.
7. Enable Discussions.
8. Publish the `v0.1.0` GitHub release.
