## GitHub release body

`Recoleta v0.1.0 is the first public baseline for a local-first AI research radar. It ingests arXiv, Hacker News, OpenReview, Hugging Face Daily Papers, and RSS, runs structured LLM analysis, and publishes the result as Markdown, trend briefs, idea briefs, and a static site.`

`This release also adds starter presets, a shorter quickstart path, and a first-output tour with sample screenshots so new users can get from clone to recognizable output faster.`
