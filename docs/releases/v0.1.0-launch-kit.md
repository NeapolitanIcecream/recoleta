# Recoleta v0.1.0 Launch Kit

Use this file when writing the first public release, a launch post, or short
outreach copy.

## Core positioning

### One-line description

`Recoleta is a local-first AI research radar that turns papers, discussion threads, and feeds into Markdown briefs and a static site.`

### Two-sentence description

`Recoleta tracks arXiv, Hacker News, OpenReview, Hugging Face Daily Papers, and RSS from one local workspace. It stores state in SQLite, writes Markdown first, and can turn the same corpus into trend briefs, idea briefs, and a GitHub Pages-ready site.`

## GitHub release body

`Recoleta v0.1.0 is the first public baseline for a local-first AI research radar. It ingests arXiv, Hacker News, OpenReview, Hugging Face Daily Papers, and RSS, runs structured LLM analysis, and publishes the result as Markdown notes, trend briefs, idea briefs, PDFs, and a static site.`

`This release also adds starter presets, a shorter quickstart path, and a first-output tour with screenshots so new users can get to recognizable output faster.`
