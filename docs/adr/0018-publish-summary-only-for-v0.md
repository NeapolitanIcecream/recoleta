## Context
Recoleta's primary user-facing artifact is a rich, evidence-first summary note.

## Decision
Published notes (Markdown/Obsidian/Telegram) include `Summary` + `Links` only.

## Consequences
Reading output becomes faster and higher-signal; LLM budget shifts to detailed summaries with quantitative evidence when available.
